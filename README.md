# Enterprise Infrastructure Website

A modern enterprise website built with React and Vite, focused on showcasing infrastructure solutions, automation systems, operational intelligence, logistics technology, and enterprise software services.

The project emphasizes premium UI design, smooth animations, responsive layouts, and high-performance front-end architecture.

---

## Features

### Hero Section

* GSAP-powered entrance animations
* Custom SVG background graphics
* Responsive typography and layout
* Modern enterprise-focused design

### Intro Section

* Scroll-triggered animations
* Structured content layout
* Premium visual presentation
* Consistent spacing system

### What We Do Section

* Sticky content panel
* Animated service cards
* Modern card-based UI
* Responsive layout

### Ecosystem Section

* Swiper.js powered image carousel
* Infrastructure and operations showcase
* Interactive slide transitions
* Mobile-friendly touch support

### Contact CTA

* Dedicated call-to-action section
* Custom SVG background graphics
* Modal-ready contact integration

### Navigation

* Fixed responsive navbar
* Hamburger menu for mobile devices
* Scroll-aware styling
* Smooth user experience

### Global Features

* Fully responsive design
* Reusable layout container system
* Smooth GSAP animations
* Interactive cursor effects
* Modern typography system

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Move into the project directory:

```bash
cd project-name
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## Required Dependencies

Core dependencies:

```bash
npm install gsap swiper
```

If ClickSpark is included:

```bash
npm install @react-bits/clickspark
```

If Tailwind CSS is used:

```bash
npm install tailwindcss @tailwindcss/vite
```
